# Luiza-Pizzaria-4bim

Relatório Trabalho Final - 4° Bimestre

Introdução

O projeto teve como objetivo o desenvolvimento de um sistema web para gerenciamento de uma pizzaria, contemplando as principais operações de um CRUD (Create, Read, Update e Delete) aplicadas às entidades Pizza, Sabor, Cliente, Funcionário, Pessoa e Pedido, além da criação de relatórios dinâmicos que exibem informações importantes de vendas.
A proposta foi construir uma aplicação completa com interface amigável, banco de dados relacional e funcionalidades automatizadas, utilizando tecnologias modernas e boas práticas de desenvolvimento.

Desenvolvimento do Projeto

Durante o desenvolvimento, foi criado um servidor Node.js com integração ao banco de dados PostgreSQL, utilizando o pacote pg para a conexão. A estrutura do projeto foi organizada em controllers, routes e views, visando manter o código limpo, reutilizável e bem estruturado.

O frontend foi desenvolvido com HTML, CSS e JavaScript puro, garantindo simplicidade e controle total da interface. Foram criadas páginas específicas para cada entidade, com formulários para cadastro, edição e exclusão, além de tabelas que exibem os dados retornados do banco.
Para os relatórios, foram geradas visualizações em formato de gráficos (pizza, barras e tabelas), mostrando os sabores mais vendidos, clientes que mais compraram e desempenho de vendas. Houve também a implementação da impressão em formato A4 para gerar relatórios físicos.

Dificuldades e Soluções

Uma das principais dificuldades encontradas foi a integração entre o backend e o frontend, especialmente no envio e recebimento de dados via API. Muitas vezes, erros de rota e configurações incorretas no servidor causavam falhas de comunicação.
Para resolver esses problemas, foi necessário estudar o funcionamento das rotas REST e ajustar a forma como os controladores retornavam as respostas em JSON. Ao decorrer do projeto teve outros pequenos desafios, sendo um deles o layout responsivo e o estilo dos relatórios, que precisaram ser adaptados para caber corretamente na folha A4 e manter o design escuro (dark theme) do sistema.

Além disso, houve um aprendizado importante sobre a estrutura MVC (Model-View-Controller) e sobre como separar corretamente as responsabilidades de cada camada, o que facilitou bastante a manutenção e evolução do sistema.

Resultados e Entregas

O projeto alcançou a maior parte dos objetivos propostos. Foi possível entregar um sistema funcional, com todas as operações de CRUD implementadas e os relatórios de vendas gerando resultados precisos.
O sistema também conta com uma interface visual organizada e coerente, incluindo um menu principal estilizado e botões de navegação.
Embora ainda haja melhorias possíveis (como autenticação de usuários e dashboard mais avançado), o resultado final é plenamente funcional e adequado para fins de aprendizado e demonstração prática.

Experiências com Inteligência Artificial

Durante o desenvolvimento, foram utilizadas as Inteligências Artificiais ChatGPT e Copilot como ferramenta de apoio técnico. As IAs auxiliaram principalmente na geração de código base, correção de erros, melhorias de estilo e explicações conceituais sobre o funcionamento de certas partes do código.
O uso da IA foi essencial para acelerar o aprendizado e otimizar o tempo de desenvolvimento, especialmente em momentos de dúvida sobre sintaxe, configuração de rotas ou estilização CSS.
O que mais se destacou foi a capacidade das IAs de gerar exemplos práticos rapidamente e ajudar a entender a lógica do backend e frontend, tornando o processo de aprendizagem mais dinâmico.

Conclusão

Durante o desenvolvimento deste projeto, pude adquirir diversos conhecimentos sobre o funcionamento de sistemas web completos, desde a estrutura de um servidor até a criação de interfaces interativas. Aprendi a importância de organizar o código em camadas (frontend, backend e banco de dados) e a utilizar ferramentas como Node.js e PostgreSQL de forma integrada.

No entanto, o que mais me chamou atenção foi a parte frontend. Gostei muito de trabalhar com HTML, CSS e JavaScript, criando as telas e ajustando os estilos para deixar o sistema mais agradável e intuitivo para o usuário. A sensação de ver o resultado visual funcionando, com botões, animações e cores combinando, foi muito satisfatória.

Apesar de ter enfrentado dificuldades com a comunicação entre o backend e o banco de dados em alguns momentos, consegui superá-las com pesquisa, prática e também com o auxílio de ferramentas de inteligência artificial, que ajudaram a identificar erros e sugerir soluções.

Em resumo, o projeto me proporcionou uma boa base em desenvolvimento web e reforçou meu interesse pela área de frontend, onde pretendo continuar me aperfeiçoando e criando interfaces cada vez mais funcionais e boas.
